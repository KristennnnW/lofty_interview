# Nexus — Telecom Intelligence Platform

B2B SaaS dashboard prototype for law enforcement telecom investigation.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Features

- **AI-native cold start** — Progressive mind map build in concentric rings
- **Activity cards** — Phone, IP, Person, Tower, Device, Event with severity classification
- **Contextual detail panel** — Raw Data, AI Insights, Timeline tabs
- **AI activity log** — Bottom-left build progress
- **AI guidance** — Appears after 15s of inactivity (demo: wait on canvas)
- **Summary pills** — Click to zoom/highlight related cards
